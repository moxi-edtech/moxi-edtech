# Glossario Operacional - Admin da Escola

Versao: 1.0.0
Data: 2026-04-03

| Termo | Definicao operacional |
|---|---|
| Admin da Escola | Perfil que governa operacao academica e administrativa da escola no portal. |
| Dashboard | Painel inicial com indicadores, alertas e atalhos de acao. |
| Radar Operacional | Bloco de alertas com pendencias criticas e recomendacao de acao. |
| KPI | Indicador de desempenho (alunos, turmas, financeiro, etc). |
| Turma pendente | Turma ainda nao validada/confirmada para uso pleno no ciclo. |
| Curriculo publicado | Estado em que configuracao academica fica valida para uso operacional. |
| Setup | Configuracao estruturante da escola (ano letivo, periodos, avaliacao, turmas). |
| Preview | Simulacao de impacto antes de gravar alteracoes finais. |
| Commit | Gravacao final e oficial das alteracoes de setup/configuracao. |
| Idempotency-Key | Chave de seguranca para evitar duplicacao de operacoes sensiveis. |
| Aluno activo | Aluno em operacao regular no sistema. |
| Aluno arquivado | Aluno retirado da operacao diaria, com possibilidade de restauracao. |
| Hard delete | Eliminacao permanente sem possibilidade de restauracao. |
| Situacao financeira | Estado financeiro do aluno (em dia, em atraso, sem registo). |
| Estado de matricula | Estado academico do aluno (matriculado, sem matricula, pendente). |
| Exportacao Excel | Ficheiro .xlsx para analise e trabalho operacional. |
| Exportacao PDF | Documento em formato PDF para consulta/arquivo oficial. |
| Auditoria | Registo de acoes realizadas por usuario no sistema. |
| Evidencia | Prova minima de que o procedimento foi executado e concluido. |
