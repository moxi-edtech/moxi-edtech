# KLASSE Network — Estratégia MED (Ministério da Educação)

**Status:** Confidencial / Estratégico  
**Fase Actual:** Fase 1 (Preparação Silenciosa)  
**Meta de Acção:** 20 escolas activas em pelo menos 2 províncias.

---

## O que estamos realmente a fazer

Não estamos a oferecer um painel. Estamos a **resolver o problema mais difícil do MED**.

O Ministério da Educação de Angola precisa de dados para tomar decisões — onde construir escolas, que disciplinas reformar, como alocar professores. Hoje esses dados chegam através de relatórios em papel que as escolas submetem anualmente, com meses de atraso, inconsistentes, e frequentemente incompletos.

Nós temos esses dados em tempo real, limpos, estruturados, e a crescer.
O KLASSE Network não é um produto. É a resposta a uma necessidade que o MED tem mas não sabe como resolver tecnicamente.

## O que o MED ganha

```text
Hoje (sem KLASSE Network):
→ Relatórios em papel
→ Dados com 6-12 meses de atraso
→ Inconsistências entre escolas
→ Zero capacidade de análise em tempo real

Com KLASSE Network:
→ Dashboard actualizado mensalmente
→ Dados de todas as escolas KLASSE num só lugar
→ Comparação por província, curso, ano lectivo
→ Análise de tendências ao longo do tempo
```

## O que a Moxi ganha (O fosso competitivo)

1. **Barreira regulatória:** Quando o MED recomenda oficialmente o KLASSE, o ónus inverte-se para qualquer concorrente.
2. **Acesso a escolas públicas:** A porta abre-se para o segmento público (10x maior).
3. **Credibilidade instantânea:** O selo de "usado pelo MED" vende escolas privadas automaticamente.
4. **Inteligência competitiva:** Saber o que reprova, onde há procura e tendências nacionais informa o nosso próprio roadmap.
5. **Irreplicabilidade:** Um concorrente com capital copia features, mas não copia anos de dados históricos e a chancela do Estado.

## O Plano de Execução

### Fase 1 — Construir os dados antes da conversa (0 → 20 escolas)
Criação de *Views* anonimizadas que não expõem IDs de escolas, alunos ou dados pessoais. Apenas dados demográficos e de performance (Província, Curso, Disciplina, Taxas de Aprovação).

### Fase 2 — Construir o painel antes da reunião (15-20 escolas)
Construção do Dashboard Interno para o MED com:
- Visão Nacional (Taxa de aprovação, Total de alunos)
- Performance por Disciplina (Ex: Matemática a 62%)
- Performance por Província
- Evolução Anual

### Fase 3 — A abordagem ao MED (20+ escolas, 2+ Províncias)
**Não vender. Oferecer.**
Apresentar o painel como uma contribuição gratuita para a monitorização estatal. O reconhecimento e a dependência tecnológica acontecem organicamente.

## Gestão de Risco
- **Privacidade:** Termos de Serviço (ToS) devem prever o uso de dados anonimizados para estatísticas de educação.
- **Independência:** O produto deve ser excepcional por si só; o acordo MED é um multiplicador, não a única fundação.
- **Timing:** Não agir antes de 20 escolas e 2 províncias. Dados fracos = proposta fraca.
