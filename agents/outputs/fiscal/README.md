# Evidencias e Outputs Fiscais

Pasta dedicada aos outputs de auditoria, validacao e inventarios do modulo fiscal.

## Convencoes

- Evidencias de producao:
  - `FISCAL_*_PROD_YYYYMMDD.(md|json)`
- Templates:
  - `templates/FISCAL_*_TEMPLATE.md`
- Relatorios historicos:
  - `INVENTARIO_*`, `RELATORIO_*`, `APPLY_DIFF_*` relacionados ao fiscal.

## Objetivo

Manter todos os artefatos fiscais de evidencia em local unico para suportar auditoria/certificacao.
