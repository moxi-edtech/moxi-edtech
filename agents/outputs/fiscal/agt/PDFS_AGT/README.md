# PDFs AGT — Convenção Operacional

Esta pasta deve conter os PDFs finais a enviar para a AGT, organizados por ponto.

## Nome recomendado

`PXX_<TIPO>_<NUMERO>_<YYYYMMDD>.pdf`

Exemplos:
- `P01_FT_FR_2026_0001_20260401.pdf`
- `P02_FT_ANULADA_2026_0002_20260401.pdf`
- `P08_FT_USD_2026_0010_20260402.pdf`

## Checklist mínimo por PDF

- Documento assinado fiscalmente
- Número fiscal legível
- Hash/HashControl consistente com o ledger
- Mensagem obrigatória AGT no documento quando aplicável
- Conteúdo correspondente ao ponto da matriz `MATRIZ_RESPOSTA_AGT_2026-03-29.md`
